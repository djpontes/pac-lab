PAC-LAB — Simulação de Algoritmos de Busca em Inteligência Artificial

📌 Descrição do Projeto

O PAC-LAB é uma aplicação desenvolvida em Python utilizando a biblioteca Pygame, com o objetivo de demonstrar de forma visual o funcionamento de diferentes algoritmos de busca em Inteligência Artificial.

O sistema simula um ambiente baseado no jogo Pac-Man, onde o usuário pode selecionar pontos de início e destino em um mapa e observar o comportamento dos algoritmos na busca por um caminho.
